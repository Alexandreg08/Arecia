# Site de Massage Bien-être

Site web pour une entreprise de massage bien-être, inspiré du concept de "Une Petite Parenthèse".

## Structure du projet

```
.
├── index.html          # Page principale
├── css/
│   └── style.css      # Styles principaux
├── js/
│   └── script.js      # Scripts JavaScript
└── README.md          # Documentation
```

## Installation

Aucune installation nécessaire. Ouvrez simplement `index.html` dans un navigateur.

## Développement

Pour tester localement, vous pouvez utiliser un serveur local :

```bash
# Avec Python
python -m http.server 8000

# Avec Node.js (si http-server est installé)
npx http-server
```

Puis ouvrez `http://localhost:8000` dans votre navigateur.
