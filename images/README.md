# Dossier Images

Placez ici l'image de fond pour la page d'accueil.

## Image requise

- **Nom du fichier** : `massage-background.jpg`
- **Format recommandé** : JPG ou PNG
- **Résolution recommandée** : 1920x1080px minimum
- **Taille recommandée** : Optimisée pour le web (< 500KB)

L'image sera utilisée comme arrière-plan de la page d'accueil.

Si l'image n'est pas présente, un dégradé de couleurs sera affiché à la place.
